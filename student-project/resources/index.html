Critical Reflection

The main difference between tools is how they run and how much control they offer. SaaS tools like Figma, Webflow, or Shopify are quick to start but rely on the cloud. Local tools like VS Code or Google Web Designer give full control and offline independence. I prefer a hybrid workflow: design in Figma, refine visuals in Webflow, and code with VS Code. This balance lets me work efficiently while maintaining creative freedom.
